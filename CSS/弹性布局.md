# 弹性布局 flex

## 1、align-content 父元素属性 - 纵向

    - flex-start
    - flex-end
    - center
    - space-between
    - space-around
    - stretch

## 2、align-self 子元素属性 - 纵向

    - flex-start  - 居上对齐
    - flex-end  - 居下对齐
    - center  - 居中对齐
    - baseline  - 基线对齐
    - stretch  - 填充容器（默认属性，仍保持自身的最大最小宽度）
